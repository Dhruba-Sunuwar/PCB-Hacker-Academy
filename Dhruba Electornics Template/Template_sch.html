How To Customize Template Files in KiCAD

This lesson demonstrates how to navigate and modify template files using the software KiCAD. It guides you step-by-step in locating KiCAD's directory, making copies of the files for customization, and returning them to a saved GitHub repository. The process explains how to modify the content within the shared folders, including inserting unique logos and text into the template. At the end of the lesson you are shown how to commit changes to the design through GitHub to create a project based on the custom template.

Lesson Overview

In the resources for the lesson, you will find the template files we used in the lesson for creating your project templates.

If you're using KiCad 7.0 and have installed it in the default Program Files directory, you can find this template file on your computer at:

C:\Program Files\KiCad\7.0\share\kicad\template

Creating the template

When you're making your own template, just follow these steps:

  1. Resize Your Logo: First, change the size of your logo to 64x64 pixels. You can do this using a free online tool like Image Resizer.

  2. Choose the Right Format: Make sure your image is saved as a ".png" file. If it's in a different format, the software might not load it into the template.

  3. Rename Your Image: Change the name of your image file to "icon.png". This way, it will show up as the icon in your User template when you start a new project.

  4. Edit the Info HTML File: Open the info HTML file with "Notepad" or any similar text editor. Then, add your own details where needed. For instance, you can change the title to something like <title> Fresu Electronics Template </title>.

By following these steps, you'll be able to create a personalized template that's ready to use.
